  • 기술만큼이나 중요한 것은 '수용성'이라고 생각합니다.


    물론 기술도 중요하지만, 지내온 시간 동안 제가 배운 것은 ‘기술보다는 소통과 태도'였습니다.

    동료와의 원활한 소통과 수용하려는 태도에서, 결국 좋은 결과물이 나온다는 믿음을 가지고 있습니다.


    변화를 유연하게 받아들이고, 동료와 담백하게 소통하며 중심을 잃지 않는 것.

    이것이 제가 지향하는 작업자의 태도입니다.


    함께 일하는 동료에게는 '말이 통하는 사람, 결과가 좋은 사람',
    그리고 사용자에게는 '직관적이고 편리한 경험을 제공해주는 사람'으로 기억되고 싶습니다.

SKILLS

  • HTML

    HTML5 웹 표준 및 웹 접근성을 준수하여 시멘틱 태그의 용도를 정확히 알고 사용할 수 있습니다.

  • CSS

    반응형 및 모바일 웹 페이지 스타일링을 할 수 있습니다.

  • SCSS

    CSS 구조를 모듈화하고, 유지보수가 용이한 스타일 시트를 작성할 수 있습니다.

  • JAVASCRIPT

    DOM 요소를 제어하고 다양한 이벤트 리스너를 적용하여 동적인 사용자 경험(UX)을 제공할 수 있습니다.

  • GIT

    소스 코드의 변경 사항을 기록하는 버전 관리 프로세스를 이해하고 있으며, 커밋 메시지 규약을 준수하여 작업 이력을 체계적으로 관리할 수 있습니다.

  • GITHUB

    원격 저장소를 활용하여 프로젝트 자산을 안전하게 관리하고, 작업 전 최신 코드 동기화(Pull) 및 결과물 공유를 위한 효율적인 협업 환경을 구축할 수 있습니다.

  • FIGMA

    피그마를 활용한 기획안을 제작할 수 있습니다.

  • PHOTOSHOP

    이미지 리터칭 및 소스 추출, 타이포그래피 도구 등을 활용해 웹 배너 및 UI 에셋을 제작할 수 있습니다.

  • VISUAL STUDIO CODE

    터미널과 Git 연동 기능을 통해, 코드 작성부터 버전 관리 작업을 수행할 수 있습니다.
